<p align="center">게시글을 포스팅하고 유저들과 소통할 수 있는 웹블로그 토이 프로젝트입니다.</p>

### 사용한 기술 스택
Java, Spring Boot, IntelliJ, Gradle, JPA, Querydsl, Redis, Nginx, Jenkins, Docker, MYSQL, ElasticSearch

### 프로젝트의 핵심 내용

* ElasticSearch로 데이터 인덱싱
* 캐싱과 인덱싱을 사용하여 조회 성능 극대화
